# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, social links, and contact information in a clean and professional UI.

## 🚀 Features

* Responsive modern UI design
* Animated typing effect using Typed.js
* Skills section
* Projects section
* Contact form integration
* Social media links
* Smooth scrolling navigation
* Mobile responsive layout
* Interactive hover effects

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Typed.js
* Boxicons
